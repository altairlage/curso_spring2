O que é o Lazy Initialization?

É a tentativa do Hibernate de carregar dados relacionados uma determinada entidade, de forma tardia ou atrasada. Assim, o Hibernate evita que sejam carregados dados desnecessariamente, carregando as listas de dados apenas quando a entidade realiza a chamada ao método get daquela lista. Por exemplo, os preços não são carregados no SQL inicial, apenas quando chamamos o método getPrecos() é que o Hibernate faz o select dos preços ligados aquele Produto.
